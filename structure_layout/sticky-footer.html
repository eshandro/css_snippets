Site Header

We're using Flexbox to make this footer be 'sticky' to the bottom of the viewport regardless of the content filling the viewport or not. Body gets display:flex; flex-flow:column; and min-height: 100vh. The Main section gets a flex: 1; This just needs to be any value greater than 0.

The catch here is that flexbox is not supported in IE before 11, and in 11 there a several bugs. There is support in Edge. There are also some bugs in FF and Chrome.
